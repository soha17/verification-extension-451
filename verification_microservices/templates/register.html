About Us

Since CCPA has gone into effect, data brokers have gained a lot of attention. They are being scrutinized for using loopholes that make it harder for users to exercise their rights. Our browser extension provides users with a a standardized verification process, so they don’t have to go through the verification process every time they attempt to exercise their CCPA rights. After verifying their identity with the browser extension, when the user submits an online request to a data broker, the user is no longer required to manually provide information for verification. Instead, the extension will inform the website that this user’s identity has already been verified. This will make it easier for users to exercise their CCPA rights as they will be able to bypass the verification process for data brokers and other technologies that request this information.
